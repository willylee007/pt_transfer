## PT转种小工具

> 纯Python工具！命令行交互式操作。支持Transmission、qBittorent 客户端。一键查找出可以转种的种子！

---

## 💻前提：**下载器已完成辅种**

## 👀原理：
1. 获取下载器的源站点种子清单
2. 获取下载器的目标站点种子清单
3. 对比目标站点没有的种子，并打印出种子详情
4. 查找出种子之后，还需用户手动在源站查找种子，并通过[一键转种](https://github.com/tomorrow505/auto_feed_js)脚本，进行转种

## 📖使用教程：
1. 添加 request 依赖，pip install requests
2. 添加下载器
3. 输入源站
4. 输入目标站
5. 输出可转种清单

## 🐱说明：
模块 2 可查询所有可转种清单，模块 3 带删除功能，仅自用
