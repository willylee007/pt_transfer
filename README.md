## PT转种小工具

> 纯Python工具！命令行交互式操作。支持Transmission、qBittorent 客户端。一键查找出本机上，可以转种的种子！

---

## 📚背景： 
本机保种客户端上有大量种子，比方说：站点 A 有大量官种，然后想要转到 C 站点，同时已通过 iyuu 对所有站点完成了辅种，这个时候还会存在很多 C 站点没有的种子。如何直接找出哪些种子还没转到 C 站点呢？于是便有了该脚本

## 💻前提：**下载器已完成辅种**

## 👀原理：
1. 获取下载器的源站点种子清单
2. 获取下载器的目标站点种子清单
3. 对比目标站点没有的种子，并打印出种子详情
4. 查找出种子之后，还需用户手动在源站查找种子，并通过[一键转种](https://github.com/tomorrow505/auto_feed_js)脚本，进行转种

## 📖使用教程：
1. 添加 request 依赖：`pip install requests`
2. 运行脚本：`python3 pt_tranfer.py`
3. 添加新下载器（录入下载器信息）
4. 选择下载器
5. 选择源站 tracker
6. 选择目标站 tracker
7. 输出可转种清单

## 🐱说明：
模块 2 可查询所有可转种清单，模块 3 带删除功能，仅自用。转种注意源站是否允许转载到其他站点，尊重各站点相关规则。
